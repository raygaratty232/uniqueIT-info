HTML теги

  • h1-h6 : тег для заголовка
  • p : тег для текста
  • a : тег для ссылки
  • img : тег для картинки
  • div : коробка , обвертка для других тегов
  • ul li : тег для списка
  • class : имя для тега, что бы писать стили только для него
  • link : подключаем стили
  • src :путь к картинке или файлу

Стили для текста

  • font-size - меняет размер текста
  • color - меняет цвет текста
  • text-align: center - текста по центру
  • line-height: - высота строки

Стили для блоков

  • width - ширина
  • height - высота
  • background-color цвет задекго фона
  • border - рамка
  • border-radius - скругляет углы
  • box-shadow: - тень
  • PX : это элемент (точка), из которого состоит изображение на экране монитора.
    В пикселях мы можем записать размер тега, например ширину 100px: width: 100px; Или размер текста: font-size: 20px;
  • Комбинации клавиш
    • ctrl + c : скопировать
    • ctrl + v : вставить
    • ctrl + z : вернуть на один шаг назад
    • ctrl + a : выделить все
    • ctrl + s : сохрать файл в SublimeText
    • ctrl + r : обновить страницу
    • Tab : SublimeText дописывает код
